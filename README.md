# TrueHire – Fake Job Post Detector

## 📌 About

TrueHire is a simple web app that helps people check if a job post is real or fake.

Many students apply for jobs online, but some jobs are fake.
They waste time and sometimes lose money.

This app helps users stay safe before applying.

## 🚀 Features

- Paste any job description
- Get risk level (Low, Medium, High)
- See score (0 to 100)
- Get reasons why a job may be fake
- Simple and easy to use

## 🛠 Built With

- Python
- Flask
- HTML
- CSS

## ⚙️ How it works

The app checks job descriptions using simple rules.

It looks for:

- “no experience required”
- “earn money fast”
- “urgent hiring”
- payment requests
- unrealistic salary
- missing company name

Based on these, it gives a score and risk level.

## ▶️ How to run the project

Clone the repository.

Go to the project folder:

```bash
cd truehire
```

Install Flask:

```bash
pip install flask
```

Run the app:

```bash
python app.py
```

Open  in your browser.
